## 常用快捷键

> 和IDA差不多

- F9 - 运行
- Ctrl F9 - 运行到返回
- F8 - 单步越过
- F7 - 单步进入
- Ctrl F2 - 重启
- Alt X - 关闭dbg

右键搜索字符串、命令等
